# 2-2. サンプル2: オブジェクトを拡大・縮小する1

## 作成するアドオンの仕様

今回作成するアドオンの使用は、以下のようにします。

* **3Dビュー** のメニュー **オブジェクト** に *選択オブジェクトの拡大* と *選択オブジェクトの縮小* を追加
* 追加した *選択オブジェクトの拡大* を実行すると、選択中のオブジェクトの大きさが2倍になる
* 追加した *選択オブジェクトの縮小* を実行すると、選択中のオブジェクトの大きさが1/2倍になる
